AIdentiti - JPN (Identity)

Identifying Anonymous Traffic

AIdentiti is a feature rich JavaScript library that is able to identify anonymous visitors, by using device fingerprints and other advanced methods.

Currently v1.0

3. All collected information is anonymous (non-personally identifiable information, Non-PII). The information is treated confidentially, and is never released to a third party. The sole purpose of collecting the data is to show the use of this passive tracking technique.
